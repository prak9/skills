# 可核验且有说服力的论证

写分析、评论、报告、方案或解释时使用本指南。目标是让读者看见结论怎样从材料中得出，而不是展示作者的草稿式思维过程。

## 目录

1. 确定主张
2. 选择证据
3. 补足推断
4. 检查因果
5. 处理反方
6. 限定结论
7. 安排正文

## 1. 确定主张

把中心问题改写成一句可以判断真假的主张。写清对象、范围、时间和评价标准。

- 宽泛：`远程办公更好。`
- 可检验：`对这支 40 人的软件团队，未来六个月保留每周两天远程办公，更可能降低离职率，且不会显著延长交付周期。`

一篇短文通常只承担一个中心主张。次级主张必须帮助证明中心主张，否则删除或另文处理。

## 2. 选择证据

按证明力选择材料：直接记录或原始数据通常强于转述；同类多例通常强于单个故事；与主张条件相近的材料强于远距离类比。

对每项关键材料说明：

- 它来自哪里，能否追溯；
- 它测量或描述了什么；
- 它没有测量或不能证明什么；
- 它与当前对象是否可比。

不要把例子当作总体证明，不要把专家身份当作证据本身，不要用无来源的“研究表明”或“普遍认为”。

## 3. 补足推断

在证据与结论之间写出读者必须知道的联系。优先说明机制、比较标准或排除过程。

基本结构可以在草稿中写成：

1. 事实或前提 P 已得到支持。
2. 若 P 成立，机制或规则 W 会产生结果 Q。
3. 当前没有更强的替代解释足以推翻 W。
4. 因此，在限定条件 C 下，可以接受结论 Q。

成稿应把这条链写成自然段落，不机械保留 P、W、Q 标签。每当正文从“发生了什么”跳到“这意味着什么”，检查中间是否缺了一步。

## 4. 检查因果

声称 A 导致 B 前，至少检查：

- A 是否先于 B；
- A 与 B 是否一起变化；
- 是否存在可解释该变化的机制；
- 共同原因 C 是否同时造成 A 和 B；
- 选择偏差、测量变化、季节性或基数效应能否解释结果；
- 反事实是否合理：没有 A 时，B 可能怎样变化。

证据不足时，把“导致”改成“与……同时发生”“可能通过……影响”或其他符合证据强度的表述，并指出仍需验证什么。

## 5. 处理反方

选择会真正改变结论的反例、替代解释或代价，不挑最弱的意见。先用对方可能接受的方式准确陈述，再作回应：

- 反方依据是否可靠；
- 它是否只适用于不同条件；
- 它揭示的是例外、代价，还是足以推翻中心主张；
- 接受它后，结论是否应缩小或改写。

承认有效限制会增强可信度。不要为了显得平衡而给证据悬殊的双方相同篇幅。

## 6. 限定结论

让措辞强度服从证据强度：

- 确证：`记录显示`、`数据表明`。
- 较强推断：`最可能的解释是`、`这些证据支持`。
- 初步假设：`一种可能机制是`、`现有材料尚不足以排除`。

写清适用范围、主要不确定性和什么新证据会改变结论。不要用层层“可能、似乎、某种程度上”表达含糊；直接命名不确定之处。

## 7. 安排正文

常用顺序如下：

1. 直接提出问题和受范围约束的主张。
2. 给出读者理解主张所需的背景。
3. 按证明力或因果顺序呈现证据。
4. 在每组证据后说明其意义，补足关键推断。
5. 处理最重要的替代解释或代价。
6. 给出带边界的结论和可执行后果。

不必给每一步加标题。短文可在数段内完成；复杂报告再用小标题帮助定位。
